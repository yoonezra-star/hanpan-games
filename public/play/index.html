How to play

한판 플레이 사용법

이 페이지는 여러 게임을 빠르게 바꿔 가며 체험할 때 쓰기 좋습니다. 특정 게임의 자세한 공략을 읽고 싶다면 각 게임의 상세 페이지로 이동하고, 바로 손을 움직이고 싶다면 이 플레이 화면에서 검색과 다시 시작 버튼만으로 흐름을 이어갈 수 있습니다.

선택

게임을 고릅니다

왼쪽 목록에서 게임 제목과 장르를 확인하거나 검색창에 이름을 입력합니다. 선택한 게임은 오른쪽 플레이 영역에 바로 로드되며, 주소에도 선택한 게임 정보가 반영됩니다.

플레이

현재 화면에서 실행합니다

게임판, 점수판, 결과 문구가 한 영역에 모여 있어 새 탭을 열지 않아도 이어서 플레이할 수 있습니다. 게임마다 버튼, 입력창, 캔버스, 카드판처럼 가장 어울리는 조작 방식을 사용합니다.

기록

결과를 보고 다시 도전합니다

가능한 게임은 최고 기록을 현재 브라우저에 저장합니다. 기록은 서버 계정과 연결하지 않으며, 같은 기기와 같은 브라우저에서 다음 도전의 기준으로 활용할 수 있습니다.

플레이 허브와 상세 페이지를 나눠 둔 이유

플레이 허브는 여러 게임을 빠르게 전환하는 공간입니다. 규칙을 이미 알고 있거나 짧게 체험하고 싶을 때는 이 화면이 가장 빠르고, 게임별 공략과 조작 설명이 필요할 때는 상세 페이지가 더 적합합니다.

상세 페이지에는 게임 방법, 점수 기준, 모바일 팁, 자주 묻는 질문을 따로 정리했습니다. 처음 보는 게임은 상세 페이지를 먼저 읽고, 익숙해진 뒤 이 플레이 허브에서 여러 게임을 이어서 즐기는 흐름을 권합니다.

플레이 허브 FAQ

이 페이지와 개별 게임 페이지는 무엇이 다른가요?

이 페이지는 여러 게임을 빠르게 선택하고 실행하는 허브입니다. 개별 게임 페이지는 한 게임의 규칙, 조작법, 점수 기준, 공략 팁을 더 자세히 읽을 수 있는 상세 페이지입니다.

게임을 선택하면 주소가 바뀌는 이유는 무엇인가요?

선택한 게임을 다시 열기 쉽도록 주소에 게임 값을 반영합니다. 예를 들어 특정 게임을 고른 뒤 주소를 공유하면 같은 게임이 열린 상태로 접근할 수 있습니다.

모바일에서도 이 플레이 허브를 사용할 수 있나요?

네. 작은 화면에서는 게임 선택 목록과 플레이 영역이 세로로 배치됩니다. 먼저 게임을 고른 뒤 아래쪽 플레이 영역에서 버튼이나 터치 조작을 사용하면 됩니다.

기록은 어디에 저장되나요?

지원되는 게임의 기록은 현재 브라우저 안에 저장됩니다. 회원 계정으로 동기화하지 않기 때문에 브라우저 데이터를 삭제하거나 다른 기기에서 접속하면 기록이 이어지지 않을 수 있습니다.

게임이 멈추거나 화면이 이상하면 어떻게 하나요?

먼저 새로고침한 뒤 같은 문제가 반복되는지 확인해 주세요. 문제가 계속되면 게임 이름, 사용한 기기, 브라우저, 재현 순서를 문의 페이지로 보내 주시면 점검에 도움이 됩니다.
