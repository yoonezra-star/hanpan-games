이 게임의 핵심

글자를 하나씩 골라 숨은 단어를 완성합니다.

행맨은 글자를 하나씩 골라 숨겨진 영어 단어를 완성하는 추리 게임입니다. 자주 나오는 모음과 글자 조합을 먼저 확인하면 남은 실패 횟수를 아끼기 쉽습니다.

게임 방법

짧은 규칙을 읽고 바로 시작할 수 있는 브라우저 미니게임입니다. 화면의 점수판과 결과 안내를 보면서 한 판씩 기록을 갱신해 보세요.

행맨은 별도 앱 설치나 로그인이 필요 없는 무료 웹게임입니다. 페이지 오른쪽 또는 아래쪽 플레이 영역에서 바로 시작할 수 있고, 한 판이 짧아 규칙을 익힌 뒤 곧바로 다시 도전하기 좋습니다.

조작과 진행

  • 알파벳 버튼을 눌러 단어에 포함된 글자를 확인합니다.
  • 맞힌 글자는 단어 위치에 공개되고, 틀린 글자는 실패 횟수로 계산됩니다.
  • 남은 실패 횟수가 없어지기 전에 단어를 완성해야 합니다.

점수와 성공 기준

  • 단어를 모두 공개하면 성공입니다.
  • 틀린 글자가 많아지면 실패에 가까워집니다.
  • 초반 글자 선택이 좋을수록 후반 후보 단어가 빠르게 줄어듭니다.

공략 팁

  • 처음 한 판은 규칙을 익히는 데 사용합니다.
  • 결과 메시지를 보고 다음 판에서 줄일 실수를 정합니다.
  • 모바일에서는 화면 버튼을 천천히 눌러 조작감을 먼저 확인합니다.
  • A, E, I, O, U 같은 모음을 먼저 확인합니다.
  • 자주 나오는 R, S, T, N 같은 자음으로 단어 윤곽을 잡습니다.
  • 공개된 위치를 보고 가능한 단어 후보를 머릿속으로 좁힙니다.

모바일 플레이 팁

  • 알파벳 버튼이 많으므로 이미 누른 글자는 다시 누르지 않도록 확인합니다.
  • 작은 화면에서는 단어 표시와 남은 실패 횟수를 번갈아 봅니다.

플레이 흐름

한 판을 시작하면 상단 점수판에서 현재 진행 상태를 확인할 수 있습니다. 결과 메시지는 실수, 성공, 최고 기록 여부를 알려 주므로 다음 판에서 무엇을 바꿀지 바로 판단할 수 있습니다.

자주 묻는 질문

기록은 어디에 저장되나요?

최고 기록은 가능한 경우 현재 브라우저 안에만 저장됩니다.

영어 단어만 나오나요?

현재 미니게임은 알파벳 기반 단어로 진행됩니다.

처음에 어떤 글자를 고르면 좋나요?

모음과 자주 나오는 자음을 먼저 고르면 단어의 구조를 빠르게 볼 수 있습니다.

모바일에서도 플레이할 수 있나요?

네. 화면 버튼과 터치 입력을 함께 고려해 작은 화면에서도 게임판과 결과 영역이 겹치지 않도록 구성했습니다.

기록이 사라질 수 있나요?

기록은 가능한 경우 현재 브라우저의 로컬 저장소에 저장됩니다. 브라우저 데이터를 삭제하거나 다른 기기에서 접속하면 이어지지 않을 수 있습니다.
